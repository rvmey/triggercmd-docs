# Windows Installation

## Download

Download the Windows agent installer:

* [TRIGGERcmdAgentSetup.exe](https://agents.triggercmd.com/TRIGGERcmdAgentSetup.exe)

## Install

1. Run the downloaded **TRIGGERcmdAgentSetup.exe** installer.
2. When prompted for a **token**, log into [triggercmd.com](https://www.triggercmd.com) and copy your token from the [Instructions page](https://www.triggercmd.com/user/computer/create).
3. Paste the token into the installer prompt.

The agent will start automatically and appear in your system tray.

## Verify

1. Log into [triggercmd.com](https://www.triggercmd.com).
2. Click **View Triggers** — you should see your computer listed with some built-in commands.
3. Click **Trigger** next to any command to test it.
