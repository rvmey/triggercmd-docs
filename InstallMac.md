# Mac Installation

## Download

Choose the correct version for your Mac:

* [Apple Silicon (M1/M2/M3/M4)](https://agents.triggercmd.com/TRIGGERcmdAgent-arm64.dmg) — most Macs from 2020 onward
* [Intel](https://agents.triggercmd.com/TRIGGERcmdAgent-x64.dmg) — older Macs

> **Tip:** Click the Apple menu → **About This Mac**. If the chip says "Apple M1" (or M2, M3, etc.), use the ARM version. If it says "Intel," use the Intel version.

## Install

1. Open the downloaded **.dmg** file.
2. Drag **TRIGGERcmdAgent** to your **Applications** folder.
3. Launch **TRIGGERcmdAgent** from Applications.
4. When prompted for a **token**, log into [triggercmd.com](https://www.triggercmd.com) and copy your token from the [Instructions page](https://www.triggercmd.com/user/computer/create).
5. Paste the token into the prompt.

The agent will appear in your menu bar.

## Verify

1. Log into [triggercmd.com](https://www.triggercmd.com).
2. Click **View Triggers** — you should see your computer listed with some built-in commands.
3. Click **Trigger** next to any command to test it.
